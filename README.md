# GateRunner

세로 오토러너 + 게이트 선택 + 로그라이트 강화 기반의 안드로이드 아케이드 게임입니다.  
Jetpack Compose(Canvas)로 구현되어 있습니다.

## 핵심 플레이
- 캐릭터는 자동으로 위로 전진
- 플레이어는 `좌/우 스와이프`로 회피 및 게이트 선택
- 구간 진행: `게이트 -> 몬스터 -> 게이트 -> 몬스터 -> 게이트 -> 몬스터 -> 보스`
- 총 3개 스테이지(숲/늪/화산), 3보스 구성
- 3스테이지 클리어 후 루프 재진입(난이도 상승)

## 전투/무기 시스템
- 무기 획득: 첫 게이트에서 무기 타입 선택
- 이후 게이트: 무기 강화 전용
- 자동 발사: 화면 내 적이 있을 때만 발사

### 무기 타입
- `멀티샷`
- `부채샷`
- `유도`
- `레이저`

### 강화 스탯
- 공격력
- 탄환 수 (레이저는 지속시간 계열로 변환)
- 공격 속도
- 공격 범위 (레이저 폭/탄환 굵기 계열)
- 관통
- 연속 발사

## 희귀도
- 노말 / 레어 / 고급 / 희귀 / 전설
- 색상:
  - 레어: 초록
  - 고급: 하늘색
  - 희귀: 보라
  - 전설: 황금

## 전설 특수 강화(무기별)
- 멀티샷/부채샷: 명중 시 소형 폭발
- 유도: 슈퍼 유도(유도탄 속도 증가 + 무한 유도)
- 레이저: 랜덤 위치 분기 레이저 6갈래 추가 발사

## 몬스터/보스
- 근접 몬스터 + 원거리 몬스터 혼합
- 스테이지별 원거리 비중 증가
- 보스 패턴은 스테이지별로 순차 로테이션
- 피격 시 데미지 숫자, 사망 이펙트 출력

## 드랍/강화 선택
- 몬스터 처치 시 확률 드랍(코인/강화 상자)
- 강화 상자 획득 시 게임 일시정지 후 3선택 강화
- 선택 UI에서 중복 강화 선택지 제거

## 메타 상점(런 시작 전)
- 코인으로 기본 스탯 영구 강화
- 코인 저장: `SharedPreferences`
- 강화 레벨이 오를수록 가격 증가

## 조작
- 게임: 스와이프 이동, 우상단 일시정지
- 일시정지: 재개 / 메인 메뉴
- 게임오버: 재시작 / 메뉴

## 실행
- IDE에서 `app` 모듈 실행 (Android Emulator 또는 실기기)
- 에뮬레이터 권장: Google APIs 이미지(API 33~35)
- 세로 모드 전용

## 저장 데이터
- 코인
- 상점 강화 레벨(공격/공속/탄환/관통/연속/범위)

## 기술 스택
- Kotlin
- Jetpack Compose
- Android Canvas Drawing
- SharedPreferences
