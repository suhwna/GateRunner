# Monster System Spec (monsters.png 기준)

## 1) 라벨 규칙
- 스프라이트 라벨: `A1 ~ ...` (행 알파벳 + 열 번호)
- 같은 행에서 좌->우, 다음 행으로 내려가면 알파벳 증가
- 이 문서는 **현재 라벨 시스템 기준**으로 패턴/체력 설계를 고정한다.

## 2) 공통 난이도 스케일
- 기준 체력(근접): `HP = 42 + stage*30`
- 기준 체력(원거리): `HP = 근접 * 0.82`
- 엘리트: `기준 * 1.9`
- 보스: `420 + stage*260`
- 루프(2회차 이상): 전체 체력 `x3.0`

- 이동속도(스크롤 보정 전 개별 이동량):
  - Stage1: `1.00x`
  - Stage2: `1.18x`
  - Stage3: `1.35x`

## 3) 패턴 정의
- `M1_직진`: 플레이어 쪽 가벼운 좌우 보정만 하며 전진
- `M2_지그재그`: 작은 사인 이동
- `M3_돌진`: 1.2~1.8초 주기 짧은 돌진
- `M4_회피`: 피격 판정 직전 좌/우 짧게 회피
- `R1_창투척`: 빠름/얇음, 직선 하향
- `R2_도끼투척`: 중속/굵음, 직선 하향
- `R3_연사투척`: 짧은 딜레이 후 2연속 투척
- `E1_브루저`: 고체력 근접 + 넓은 히트박스
- `E2_캐스터`: 저이동 + 원거리 고빈도
- `B1_보스탄막`: 플레이어 방향 3연 발사
- `B2_좌우레이저`: 좌/우 반면 경고 후 레이저
- `B3_폭발탄`: 느리게 전진하는 대형 탄

## 4) 스테이지별 몬스터 편성 (라벨 고정)

## Stage 1 (숲)
- 근접 풀: `A3, A4, A5`
  - A3: `M1_직진`
  - A4: `M2_지그재그`
  - A5: `M3_돌진`
- 원거리: `A2`
  - A2: `R1_창투척` (70%), `R2_도끼투척` (30%)
- 엘리트: `B1`
  - B1: `E1_브루저`
- 보스: `A10`
  - A10: `B1_보스탄막 + B2_좌우레이저`

## Stage 2 (늪)
- 근접 풀: `C1, C2, C3, C4`
  - C1: `M2_지그재그`
  - C2: `M4_회피`
  - C3: `M2_지그재그 + M4_회피`
  - C4: `M3_돌진`
- 원거리: `C6, D1`
  - C6: `R2_도끼투척`
  - D1: `R3_연사투척`
- 엘리트: `D4`
  - D4: `E2_캐스터`
- 보스: `D6`
  - D6: `B1_보스탄막 + B3_폭발탄`

## Stage 3 (화산)
- 근접 풀: `E2, E3, E4, E5`
  - E2: `M3_돌진`
  - E3: `M3_돌진 + M4_회피`
  - E4: `M2_지그재그(강)`
  - E5: `M1_직진(고속)`
- 원거리: `E7, F2`
  - E7: `R1_창투척(고속)`
  - F2: `R3_연사투척(고빈도)`
- 엘리트: `F4`
  - F4: `E1_브루저 + R2_도끼투척`
- 최종보스: `F6`
  - F6: `B1_보스탄막 + B2_좌우레이저 + B3_폭발탄`

## 5) 패턴 수치(초기값)
- `M2_지그재그`: 진폭 22 / 주기 0.9s
- `M3_돌진`: 주기 1.4s, 돌진시간 0.28s, 속도 +180%
- `M4_회피`: 쿨 0.7s, 횡이동 22px
- `R1_창투척`: 속도 15.5
- `R2_도끼투척`: 속도 11.5, 판정반경 +35%
- `R3_연사투척`: 발사간격 180ms, 2연사
- `B1_보스탄막`: 3발 x 3회, 회차간격 260ms
- `B2_좌우레이저`: 경고 700ms, 지속 520ms
- `B3_폭발탄`: 속도 6.2, 반경 2.1x

## 6) 구현 메모
- Stage1은 사용자 지정사항 유지:
  - 원거리 `A2` 고정
  - 근접 `A3/A4/A5` 랜덤
- 이후 Stage2/3 라벨도 동일한 방식으로 `spriteRawIndex` 고정 풀에서 랜덤 선택 권장
- 표시용 디버그 라벨은 빌드 전 제거 가능
